
As soon as the zip file is extracted please run the bellow commands on terminal

$ pip install -r requirements.txt 

$ python manager.py runserver

The first command will install all the required python packages to run the web app

The second commmand will run the server, please visit http://127.0.0.1:8000/ to launch the web app

As the project is built on django and sqlite, the database is built in and does not require any additional software or workbench to be downloaded.

At any point in time if one wants to see the database please visit http://127.0.0.1:8000/admin and use the below credentials
